# Contribution

There is a great need for community support to adapt the TIS rules, as individuals do not have the energy to manage so many rules.

## Contributors

## New rules

You can add a brick to the TIS tweak rules.

### Prerequisites

- Github
- Git

### How to contribute

1. Fork the project and create a branch

2. Create a new tweak rule according to "[Rule Syntax](/advanced/ruleSyntax)" and add it to the `/tweaks` folder

3. `git push` then go to [TIS project Github](https://github.com/whatk233/tis) and create a pr (pull request) and wait for the merge! ðŸ˜Š If you have any questions, a member of the project will discuss it with you in the pr.

## Question feedback

Rules not working? Rule has side effects? Rule is causing some system functions to be abnormal?

Create an "[Issues](https://github.com/whatk233/tis/issues)" to let us know about the issue and a developer or volunteer will discuss it with you
