## 环境

- Deno (1.29.3 或更高)
- [Autoit Compiler Toolkit](https://github.com/whatk233/setup-autoit-compiler-toolkit/releases/download/toolkit/autoit_compiler_toolkit_v3.3.16.1_230116.zip)
- Windows 操作系统 (编译 TIS Core 需要)

## 配置开发环境

1. 安装 Deno
2. 下载 Autoit Compiler Toolkit 并解压
3. 将 Autoit Compiler Toolkit 设置为系统/环境变量，变量名为 `AUTOIT_COMPILER_TOOLKIT`

## 编译 TIS Core

执行 `deno task build`，编译好的二进制文件、配置文件会存放在 `/build` 文件夹中
